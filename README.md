## What is AIHub Browser?

AIHub Browser is a next-generation **AI-native web browser** built on Electron + Chromium. Instead of bolting AI on as an extension, the AI assistant is first-class — built directly into the browser shell alongside a 3D bookmark knowledge graph, local Ollama AI support, and a built-in VPN proxy.

It is what you get if Chrome, Arc, Obsidian, and ChatGPT merged into one application.

## Getting Started

### 1. Download & Install

Run `AIHub-Browser-Setup-1.0.0.exe`. The installer sets up the browser and optionally installs Ollama for private local AI.

### 2. Configure AI

Open **Settings → AI Configuration** and either:

- Paste an [OpenRouter](https://openrouter.ai) API key (free tier available)
- Point to a local Ollama instance (`http://localhost:11434`)

### 3. Browse Smarter

- Press `Ctrl+Shift+A` to open the AI panel on any page
- Click the sphere icon in the sidebar to open the 3D bookmark graph
- Ask the AI assistant for live news, summaries, or research
