# Claude.md — AIHub Browser (Next-Generation AI Browser)

## Project Overview

AIHub Browser is a next-generation AI-native web browser that combines:

- AI-powered browsing and research
- Visual bookmark management
- Obsidian knowledge integration
- Multi-model AI assistance
- Privacy-first architecture
- Intelligent workspace organization
- Agentic automation
- Modern cross-platform user experience

The browser should feel like the future evolution of Chrome, Arc, Edge, Obsidian, Cursor, and ChatGPT combined into a single intelligent environment.

---

# Vision

Build the world's most intelligent AI browser where:

- Every tab becomes knowledge.
- Every bookmark becomes connected intelligence.
- Every website can be summarized, analyzed, and automated.
- AI agents can assist users while browsing.
- Research becomes effortless.
- Users can interact with multiple AI models simultaneously.

AIHub Browser is not just a browser.

It is an AI Operating System for the Web.

# Claude Development Instructions

When developing AIHub Browser:

1. Write production-quality code.
2. Use SOLID principles.
3. Use clean architecture.
4. Avoid mock functionality unless requested.
5. Build modular components.
6. Create reusable services.
7. Prioritize performance.
8. Prioritize security.
9. Create beautiful UI/UX.
10. Make the application feel handcrafted, not AI-generated.

Every feature should be scalable, maintainable, and enterprise-ready.

The final experience should rival or exceed Chrome, Arc Browser, Edge, and other modern AI browsers.
